# OWT - Ovsb Widget Toolkit ♥

OWT é a biblio gráfica nativa do Ovsb.OS, agr com WM integrado!

**Widgets**
- Window, Label, Button, TextBox, TextArea
- ListView, ComboBox, Menu, Dialog, StatusBar
- Tema escuro/claro, primitivas de desenho

**WM** (Window Manager)
- Backbuffer pra desenhar sem flicker
- flush pro framebuffer do video
- Screen info pra resolucao dinamica
